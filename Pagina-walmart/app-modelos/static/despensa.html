Este módulo utiliza información derivada del archivo dataset_walmart.csv, el cual contiene registros históricos de compras realizadas por clientes en diferentes sucursales comerciales.

Para el desarrollo del modelo se seleccionaron variables relacionadas con las características del hogar y hábitos de consumo, debido a que estos factores influyen directamente en la cantidad y tipo de productos adquiridos.

Las variables utilizadas fueron:

  • Número de personas: permite estimar el volumen de consumo familiar.
  • Tamaño del hogar (m²): se relaciona con necesidades de limpieza y abastecimiento.
  • Tipo de genero (H, M, H-M): representa patrones generales de consumo asociados al hogar.

Los datos son de naturaleza estructurada y descriptiva, ya que representan patrones de compra y comportamiento comercial. A partir de aproximadamente 79,875 registros históricos, el sistema aplica un modelo basado en K-Nearest Neighbors (KNN) para identificar perfiles similares y generar recomendaciones inteligentes de productos para despensa y hogar.
