description Descripción, naturaleza y volumen de los datos

Este módulo utiliza información histórica proveniente del archivo dataset_walmart.csv, el cual contiene registros comerciales asociados a clientes, productos, horarios y métodos de pago.

Para el análisis de comportamiento de clientes se seleccionaron variables relacionadas con características de compra y segmentación comercial, debido a que permiten identificar patrones de consumo y tendencias de comportamiento dentro de las sucursales.

Las variables utilizadas fueron:

  • Gender: utilizado para segmentar comportamiento por género.
  • Customer Type: permite clasificar clientes por tipo de consumo.
  • Invoice ID: empleado para contabilizar clientes únicos y compras realizadas.
  • Hora: utilizada para identificar horarios críticos y picos de concurrencia.
  • Producto: permite detectar artículos más comprados por cada segmento.
  • Método de pago: utilizado para identificar preferencias de compra.
  • Sucursal: permite comparar comportamiento entre distintos puntos de venta.

Los datos son de naturaleza estructurada, descriptiva y comercial, ya que representan hábitos de consumo y comportamiento de clientes dentro del dominio empresarial. El sistema procesa aproximadamente 79,875 registros históricos y cerca de 18,294 clientes únicos para construir grupos de comportamiento, detectar horarios críticos y analizar patrones de compra.
