Este módulo utiliza información proveniente del archivo dataset_walmart.csv, el cual contiene registros históricos de compras realizadas por clientes en distintas sucursales comerciales.

Para el análisis de recomendaciones se seleccionaron variables relacionadas con productos, compras y precios, debido a que permiten identificar patrones de asociación y comportamiento de consumo entre artículos adquiridos en conjunto.

Las variables utilizadas fueron:

  • Producto: utilizado para identificar relaciones de compra entre artículos.
  • Invoice ID: empleado para detectar compras compartidas dentro de una misma transacción.
  • Precio unitario: utilizado para mostrar información económica asociada a las recomendaciones.
  • Sucursal: permite analizar comportamiento comercial por punto de venta.

Los datos son de naturaleza estructurada y transaccional, ya que representan operaciones comerciales registradas históricamente. El sistema analiza aproximadamente 79,875 registros mediante modelos de asociación de productos para generar recomendaciones inteligentes orientadas al análisis comercial y apoyo en la toma de decisiones.
